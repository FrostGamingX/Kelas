Belajar HTML dan CSS

HTML digunakan untuk membuat tampilan website. CSS digunakan untuk mengatur tampilan.

Materi HTML

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untuk tabel
  • input untuk form
  • li untuk list/daftar

Materi CSS

  1. margin, padding, position untuk mengatur jarak antara element HTML
  2. display untuk mengatur element HTML agar tampil/tidak tampil, mendatar (kolom)/menurun (baris)
  3. style untuk mengatur font
